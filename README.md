# AURA – AI Study Assistant 🤖📚

AURA is an autonomous AI-powered learning assistant designed to support and optimize a student’s study journey.  
It goes beyond simple Q&A by actively planning, tracking, and improving learning through intelligent automation.

---

## 🚀 Key Features

- 🧠 **Multi-Agent AI System**
  - Coordinator: Orchestrates multiple agents and manages task execution
  - Daily Planner: Adapts study plans based on strengths and weaknesses
  - Recall Engine: Uses the forgetting curve to schedule revisions
  - Resource Hunter: Retrieves relevant learning materials using semantic search

- ⚡ **Real-Time Learning Optimization**
  - Dynamic study plans based on performance
  - Personalized dashboard with progress tracking and insights
  - Exam countdown and mastery scoring

- 📊 **Proactive Learning Experience**
  - Tracks topics, performance, and study streaks
  - Provides continuous AI-driven recommendations

---

## 🧩 System Architecture

- Multi-agent architecture coordinated by a central orchestrator  
- Parallel processing of learning tasks  
- Event-driven updates based on user performance  

---

## 🔍 Knowledge & Retrieval (RAG)

- Uses a Retrieval-Augmented Generation (RAG) pipeline  
- Embedded documents stored in a vector database  
- Indexed 1000+ learning resources (notes, textbooks, past papers)  
- Enables fast and accurate knowledge retrieval
